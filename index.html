Ejercicio 2

        

          Haga una funcion que solo reciba arreglos con mas de 5 elementos y
          luego elimine los ultimos 3.
        

        

          En caso de recibir un arreglo de menos de 5 elementos, mostrar en
          pantalla un mensaje de error.
        

        
Entrada valida: [1, 2, 3, 4, 5, 6, 7]

        
Resultado esperado: [1, 2, 3, 4]


        
Entrada invalida: [1, 2, 3, 4, 5]

        

          Resultado esperado:
          Error: el arreglo debe tener mas de 5 elementos
