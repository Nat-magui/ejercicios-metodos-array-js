# Ejercicios JavaScript - Métodos de Array

Proyecto práctico desarrollado con **HTML**, **CSS** y **JavaScript** para resolver ejercicios enfocados en métodos de arrays, funciones y lógica de programación.

La idea del proyecto es presentar cada ejercicio en una página web simple e interactiva, donde se puede ejecutar cada solución y visualizar el resultado en pantalla.

## Objetivo

Practicar conceptos fundamentales de JavaScript, especialmente:

- Arrays
- Funciones
- Métodos mutables e inmutables
- Métodos de búsqueda
- Métodos de iteración
- Manipulación de datos
- Uso básico del DOM

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Ejercicios incluidos

1. Eliminar el primer elemento de un arreglo.
2. Eliminar los últimos 3 elementos si el arreglo tiene más de 5 elementos.
3. Sumar un número a cada elemento de un arreglo.
4. Crear un arreglo desde `1` hasta `n`.
5. Encontrar el número distinto dentro de un arreglo.
6. Encontrar la menor suma posible entre números enteros.
7. Encontrar el mayor producto entre dos números adyacentes.
8. Advertir qué oveja está cerca del lobo.

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en el navegador.
3. Presionar los botones de cada ejercicio para ver el resultado.

## Funcionalidades

* Cada ejercicio está presentado en una card.
* Cada card incluye el enunciado, entrada de ejemplo y resultado esperado.
* Los botones ejecutan las funciones correspondientes.
* Los resultados se muestran directamente en pantalla.
* La lógica de los ejercicios está separada de la lógica del DOM.

## Aprendizaje principal

Este proyecto me permitió practicar cómo resolver ejercicios de lógica con JavaScript y cómo organizar el código en archivos separados para que sea más claro, mantenible y fácil de entender.

También me ayudó a reforzar la diferencia entre métodos que modifican el array original y métodos que devuelven un nuevo array.
